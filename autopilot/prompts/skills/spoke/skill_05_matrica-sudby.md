## Что делает эту серию уникальной

Матрица судьбы (метод Александрова, известен также как «квадрат Пифагора» и
«психоматрица») строится из цифр даты рождения, разложенных в сетку 3×3.
Серия покрывает два типа страниц под одним шаблоном — тип определяется
полем `unit_type` в карточке:

- **`cell`** — одна цифра матрицы (1-9) и её количество в сетке
- **`line`** — одна из восьми линий (3 столбца, 3 строки, 2 диагонали)

Оба типа читаются по одной логике: недостаток / норма / избыток —
поэтому у них общая структура блоков, но разное содержание.

Два обязательных уникальных блока на каждой странице серии:
- **Карточка ячейки/линии** — fact_row с 5 полями, уникальна для каждой страницы
- **Проявления в жизни** — table с 5 строками, уникальна для каждой страницы

## Методика расчёта (для вводных абзацев и H2 "Как считается")

Агент не пересчитывает даты сам — методика нужна только чтобы корректно
объяснять её читателю в тексте.

**Рабочие числа** — четыре производных числа от даты ДД.ММ.ГГГГ:

```
a = сумма всех цифр даты                     (22.03.1990 → 2+2+0+3+1+9+9+0 = 26)
b = сумма цифр числа a                        (2+6 = 8)
c = a минус удвоенная первая цифра даты       (26 − 2×2 = 22)
d = сумма цифр числа c                        (2+2 = 4)
```

Все цифры даты + a + b + c + d объединяются в один набор, и по нему
считается, сколько раз встречается каждая цифра 1-9. Ноль в подсчёт
не идёт — своей ячейки в матрице у него нет.

**Сетка 3×3:**
```
1  4  7
2  5  8
3  6  9
```
Количество повторений цифры записывается в её ячейку (например «111»
для тройки, если она встретилась 3 раза).

**Восемь линий** — три столбца, три строки, две диагонали:

| Линия | Позиции | Сфера |
|---|---|---|
| Столбец 1 | 1-4-7 | Целеустремлённость, воля |
| Столбец 2 | 2-5-8 | Семья и деньги |
| Столбец 3 | 3-6-9 | Талант, интуиция, оптимизм |
| Строка 1 | 1-2-3 | Темперамент |
| Строка 2 | 4-5-6 | Хозяйственность, материальное |
| Строка 3 | 7-8-9 | Чувство долга, сознание |
| Диагональ 1 | 1-5-9 | Цель, притяжение |
| Диагональ 2 | 3-5-7 | Юмор, гибкость |

⚠️ Названия линий — синтез наиболее часто встречающихся трактовок,
не единый общепринятый стандарт (в отличие от чисел судьбы 1-9, где
трактовки между источниками совпадают гораздо стабильнее). Это
канон именно для этого сайта — не менять между статьями.

**Особый случай — рождённые в 2000-2019 годах:** из-за структуры года
(20XX) многие ячейки закономерно остаются пустыми. Упоминать это стоит
на страницах цифр 1, 2 и линий с ними связанных (столбец 1-4-7,
строка 1-2-3) — там пустая ячейка у молодых читателей особенно частый
случай. Не преподносить пустую ячейку как «поломанную дату» — это
особенность метода, не ошибка.

## Структура H2

### Обязательные H2 — присутствуют на каждой странице серии:

- **Что означает [цифра/линия] в матрице судьбы** — суть, место в системе
- **Как [цифра/линия] проявляется в характере и жизни** — практическое чтение
- **Часто задаваемые вопросы** — 2-3 вопроса из семантики

### Вариативные H2 — агент выбирает 2-3 под конкретную ячейку/линию:

| H2 | Когда использовать |
|---|---|
| Если [X] нет в матрице | все страницы — самый частый практический вопрос |
| [X] в избытке — что это значит | все страницы |
| [X] в работе и карьере | 1, 4, 7, 8 и связанные линии — числа действия |
| [X] и здоровье | цифра 4 (буквально сфера здоровья), линия 4-5-6 |
| [X] и отношения | 2, 5, 6 и связанные линии — числа связи |
| [X] у детей 2000-х годов рождения | 1, 2, столбец 1-4-7, строка 1-2-3 |
| Как усилить [X] | все страницы — практический раздел |
| [X] и деньги | 8, столбец 2-5-8 |

Выбирай H2 которые органичны именно этой цифре/линии — не добавляй ради объёма.

### H3 под каждым H2:

Агент определяет самостоятельно, 2-3 под каждым H2, не копировать между страницами серии.

## Блок "Карточка ячейки/линии"

**Место:** сразу после lead первого H2, до первого H3.
**Тип блока:** `fact_row`
**Назначение:** читатель сразу видит суть цифры/линии и три её состояния.

Пять обязательных полей:

| Поле | Что писать |
|---|---|
| Сфера | к какой стороне жизни относится цифра/линия |
| Если пусто | что означает отсутствие (0 повторений / пустая линия) |
| Если слабо выражено | 1-2 повторения — умеренное проявление |
| Если в избытке | 3+ повторения — гипертрофированное проявление |
| Символ | образ цифры/линии в одном слове |

Содержание уникально для каждой страницы — не копировать между цифрами/линиями.

Примеры:

**Цифра 1 (характер и воля):**
```
Сфера → Характер и воля
Если пусто → нерешительность, зависимость от чужого мнения, слабая инициатива
Если слабо выражено → здоровая уверенность, умеренная воля
Если в избытке → эгоцентризм, потребность доминировать, деспотичность
Символ → Стержень
```

**Линия 1-4-7 (целеустремлённость):**
```
Сфера → Целеустремлённость и воля
Если пусто → сложно ставить цели и идти к ним без внешней опоры
Если слабо выражено → цели есть, но реализация нестабильна
Если в избытке → одержимость целью, риск идти по головам
Символ → Стрела
```

```json
{
  "type": "fact_row",
  "items": [
    {"label": "Сфера", "value": "уникально под цифру/линию"},
    {"label": "Если пусто", "value": "уникально под цифру/линию"},
    {"label": "Если слабо выражено", "value": "уникально под цифру/линию"},
    {"label": "Если в избытке", "value": "уникально под цифру/линию"},
    {"label": "Символ", "value": "уникально под цифру/линию"}
  ]
}
```

## Блок "Проявления в жизни"

**Место:** сразу после lead второго обязательного H2, до первого H3.
**Тип блока:** `table`
**Назначение:** читатель видит, как цифра/линия работает в разных сферах жизни.

Три колонки, пять строк:

| Сфера жизни | Как проявляется в избытке/силе | Как проявляется в недостатке |
|---|---|---|
| Характер и решения | ... | ... |
| Отношения | ... | ... |
| Работа и карьера | ... | ... |
| Здоровье и энергия | ... | ... |
| Духовный путь | ... | ... |

Правила:
- Содержание уникально для каждой цифры/линии
- Не оценочно ("хорошо/плохо"), а описательно — суть проявления
- Для линий колонки читаются как "сильная линия" / "слабая линия" вместо "избыток/недостаток"

```json
{
  "type": "table",
  "caption": "Проявления [цифры/линии] в разных сферах жизни",
  "headers": ["Сфера жизни", "Как проявляется в избытке/силе", "Как проявляется в недостатке"],
  "rows": [
    ["Характер и решения", "...", "..."],
    ["Отношения", "...", "..."],
    ["Работа и карьера", "...", "..."],
    ["Здоровье и энергия", "...", "..."],
    ["Духовный путь", "...", "..."]
  ]
}
```
