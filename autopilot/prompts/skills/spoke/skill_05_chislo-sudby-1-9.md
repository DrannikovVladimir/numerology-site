## Что делает эту серию уникальной

Каждая страница серии — портрет одного числа как живой сущности.
Числа 1-9 принципиально разные по характеру, энергетике и жизненной миссии.
Структура статьи строится под конкретное число — не копируется между страницами.

Два обязательных уникальных блока на каждой странице серии:
- **Числовой портрет** — карточка с 5 полями, уникальна для каждого числа
- **Матрица совместимости** — таблица 9 строк, уникальна для каждого числа

## Структура H2

### Обязательные H2 — присутствуют на каждой странице серии:

- **Значение числа [N] в нумерологии** — суть, энергетика, место в системе
- **Число [N] в любви и отношениях** — совместимость, поведение в паре
- **Часто задаваемые вопросы** — 2-3 вопроса из семантики

### Вариативные H2 — агент выбирает 2-3 под характер конкретного числа:

| H2 | Для каких чисел органичен |
|---|---|
| Люди с числом судьбы [N] | все числа |
| Число [N] в карьере и деньгах | 1, 4, 8 — числа действия |
| Число [N] и творчество | 3, 6 — творческие числа |
| Число [N] и духовный путь | 7, 9, 11 — числа поиска |
| Число [N] и власть | 1, 8 — числа лидерства |
| Число [N] и служение | 2, 6, 9 — числа отдачи |
| Число [N] — практика и применение | все числа |
| Число [N] и деньги | 4, 8 — практические числа |

Выбирай те H2 которые раскрывают природу конкретного числа наиболее точно.
Не добавляй H2 ради объёма — только если есть что сказать по существу.

### H3 под каждым H2:

Агент определяет H3 самостоятельно исходя из содержания H2 и природы числа.
2-3 H3 под каждым H2 — как в hub-скиле.
Не копировать H3 между разными числами.

## Блок "Числовой портрет"

**Место:** сразу после lead первого H2, до первого H3.
**Тип блока:** `fact_row`
**Назначение:** визуальная карточка числа — читатель сразу видит суть.

Пять обязательных полей:

| Поле | Что писать |
|---|---|
| Кармический потенциал | дар числа, что оно несёт в мир |
| Теневая сторона | главная слепая зона, тень числа |
| Ключевой урок | что нужно освоить в этой жизни |
| Символ | образ числа в одном слове |
| Мастер-связь | с каким мастер-числом резонирует и почему коротко |

Содержание уникально для каждого числа — не копировать между статьями.

Примеры:

**Число 1:** Факел / Одиночество / Доверять другим / Первопроходец / 11 — удвоенная воля
**Число 2:** Мост / Растворение / Говорить «нет» / Дипломат / 22 — строитель мира
**Число 7:** Маяк / Изоляция / Открываться людям / Мудрец / 33 — мудрость на службе

```json
  ]
}
```

## Блок "Матрица совместимости"

**Место:** сразу после lead H2 про отношения, до первого H3.
**Тип блока:** `table`
**Назначение:** читатель сразу видит характер пары со всеми числами.

Три колонки, девять строк — по одной на каждое число от 1 до 9.
Содержание уникально для каждой статьи — таблица числа 1 и числа 7 разные.

| Число | Характер пары | Ключевой вызов |
|---|---|---|
| 1 | одна фраза — суть взаимодействия | одна фраза — главное напряжение |
| ... | ... | ... |

Правила:
- Строка с самим числом (например, 7 и 7) — описывает пару одинаковых чисел
- "Характер пары" — нейтрально, не "хорошо/плохо", а суть динамики
- "Ключевой вызов" — конкретная задача пары, не оценка

```json
{
  "type": "table",
  "caption": "Совместимость числа [N] с другими числами судьбы",
  "headers": ["Число", "Характер пары", "Ключевой вызов"],
  "rows": [
    ["1", "...", "..."],
```
