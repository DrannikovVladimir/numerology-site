## Что делает эту серию уникальной

Каждая страница — разбор конкретного времени как нумерологического сигнала.
Читатель приходит с конкретным вопросом: "я только что увидел ЧЧ:ММ — что это значит?"
Структура отвечает на этот вопрос с разных сторон жизни.

Контент уникален потому что:
- Каждое время имеет свою числовую сумму и энергетику
- Контекст появления разный — 00:00 и 11:11 люди замечают в разные моменты жизни
- Тип времени (зеркальное / палиндром / прочее) меняет угол подачи

Два обязательных уникальных блока на каждой странице серии:
- **Карточка времени** — fact_row с 6 полями, уникальна для каждого времени
- **Время и жизненные ситуации** — table с 5 строками, уникальна для каждого времени

## Классификация типов времён

Агент определяет тип времени из URL страницы (например /numerologiya-na-chasakh/11-11/).

### Тип 1 — Зеркальные (ЧЧ:ЧЧ)
Примеры: 00:00, 01:01, 11:11, 22:22, 23:23

Одно число повторяется дважды — энергия числа усилена вдвое.
Самый сильный нумерологический сигнал среди всех типов.
Акцент на вибрации числа, его послании и призыве к действию.

### Тип 2 — Палиндромы (ЧЧ:ЧЧ' где минуты = зеркало часов)
Примеры: 12:21, 13:31, 20:02, 05:50, 14:41

Время читается одинаково слева направо и справа налево.
Тема: симметрия, баланс, возврат к себе, отражение.
Число часов задаёт основную тему, минуты — её отражение и завершение.

### Тип 3 — Прочие (ЧЧ:ММ где числа разные и не зеркало)
Примеры: 11:22, 22:33, 02:22, 05:55, 11:44, 22:11

Два разных числа в диалоге — взаимодействие энергий.
Число часов задаёт основную тему и контекст.
Число минут усиливает, уточняет или добавляет второй план.

## Числовая энергетика

### Как считать сумму времени

Складываем все 4 цифры времени:
```
11:11 → 1+1+1+1 = 4
22:22 → 2+2+2+2 = 8
12:21 → 1+2+2+1 = 6
```

Если сумма больше 9 — редуцируем до однозначного числа (кроме мастер-чисел).

### Мастер-времена — особое правило

Времена 11:11 и 22:22 читаются как мастер-числа 11 и 22 —
несмотря на то что их простая сумма 4 и 8.
Причина: само время содержит зеркально повторяющееся мастер-число.

```
11:11 → простая сумма 4, но читается как мастер-11
22:22 → простая сумма 8, но читается как мастер-22
```

### Таблица сумм и энергетики

| Сумма | Времена | Энергетика |
|---|---|---|
| 0 | 00:00 | пустота, обнуление, чистый лист |
| 1 | 05:05, 14:14, 23:23, 05:50, 14:41, 23:32, 22:33, 11:44, 13:33 | старт, инициатива, новый цикл |
| 2 | 01:01, 10:10, 19:19, 10:01 | баланс, интуиция, партнёрство |
| 3 | 06:06, 15:15, 15:51, 22:44, 11:55, 04:44 | творчество, радость, общение |
| 4 | 02:02, 11:11*, 20:20, 20:02, 14:44 | структура, стабильность, труд |
| 5 | 07:07, 16:16, 22:55 | перемены, свобода, движение |
| 6 | 03:03, 12:12, 21:21, 12:21, 21:12, 11:22, 22:11, 02:22, 22:02, 05:55 | гармония, забота, ответственность |
| 7 | 08:08, 17:17 | мудрость, духовность, поиск |
| 8 | 04:04, 13:13, 22:22*, 13:31, 04:40, 11:33 | сила, материальный мир, власть |
| 9 | 09:09, 18:18, 03:33 | завершение, трансформация, отпускание |
| мастер-11 | 11:11 | интуиция, синхроничность, портал, миссия |
| мастер-22 | 22:22 | строитель, воплощение идей, мастерство |

*Простая сумма 4 и 8 — но читаются как мастер-числа (см. правило выше).

## Структура H2

### Обязательные H2 — присутствуют на каждой странице серии:

- **Что означает [ЧЧ:ММ] на часах** — суть послания, числовая энергетика
- **[ЧЧ:ММ] в любви и отношениях** — самый популярный запрос серии
- **Часто задаваемые вопросы** — 2-3 вопроса из семантики

### Вариативные H2 — агент выбирает 2-3 по типу и энергетике времени:

| H2 | Когда использовать |
|---|---|
| Почему вы видите [ЧЧ:ММ] | все типы — объяснение феномена |
| [ЧЧ:ММ] и духовный путь | сумма 7, 9, 11, 22, 33 или тип "зеркальное" |
| [ЧЧ:ММ] в карьере и деньгах | сумма 1, 4, 8 — числа действия |
| [ЧЧ:ММ] и новые начинания | сумма 1, 0 — числа старта |
| [ЧЧ:ММ] и завершение цикла | сумма 9 — число завершения |
| Что делать когда видишь [ЧЧ:ММ] | все типы — практический раздел |
| [ЧЧ:ММ] как знак синхроничности | 11:11, 22:22, зеркальные с мастер-суммой |
| [ЧЧ:ММ] — послание двух чисел | тип "прочее" — разбор диалога двух чисел |
| Симметрия времени [ЧЧ:ММ] | тип "палиндром" — тема отражения и баланса |
| [ЧЧ:ММ] в разных системах нумерологии | особые времена — 11:11, 22:22, 00:00 |

### H3 под каждым H2:

Агент определяет H3 самостоятельно под тип времени и его энергетику.
2-3 H3 под каждым H2.
Не копировать H3 между разными временами серии.

## Блок "Карточка времени"

**Место:** сразу после lead первого H2, до первого H3.
**Тип блока:** `fact_row`
**Назначение:** читатель сразу видит суть этого времени.

Шесть обязательных полей:

| Поле | Что писать |
|---|---|
| Числовая сумма | к чему сводится ЧЧ:ММ (например 11:11 → 4 или мастер-11) |
| Тип сигнала | зеркальный / палиндром / диалог чисел |
| Энергия момента | суть вибрации этого времени в 3-5 словах |
| Типичный контекст | когда люди чаще всего замечают именно это время |
| Послание | ключевое сообщение в одном предложении |
| Как реагировать | конкретное действие или намерение |

Содержание уникально для каждого времени — не копировать между статьями.

Примеры:

**00:00:**
```
Числовая сумма → 0 — точка обнуления
Тип сигнала → зеркальный
Энергия момента → переход, пустота, чистый лист
Типичный контекст → полночь, момент перехода дня, усталость или бессонница
Послание → всё завершилось — можно начать заново
Как реагировать → загадайте намерение на следующий цикл
```

**11:11:**
```
Числовая сумма → мастер-число 11
Тип сигнала → зеркальный, особый
Энергия момента → интуиция, синхроничность, портал
Типичный контекст → случайный взгляд на телефон, момент важной мысли
Послание → вселенная подтверждает — вы на правильном пути
Как реагировать → зафиксируйте мысль которая была в момент взгляда
```

**12:21:**
```
Числовая сумма → 6 — гармония
Тип сигнала → палиндром
Энергия момента → симметрия, возврат к себе, завершение полукруга
Типичный контекст → обеденное время, середина дня
Послание → то что ушло — возвращается в новой форме
Как реагировать → проверьте что из начатого утром требует внимания
```

## Блок "Время и жизненные ситуации"

**Место:** сразу после lead H2 про любовь и отношения, до первого H3.
**Тип блока:** `table`
**Назначение:** читатель видит как это время работает в разных контекстах жизни.

Три колонки, пять строк:

| Ситуация | Что означает это время | Как использовать |
|---|---|---|
| Любовь и отношения | ... | ... |
| Карьера и деньги | ... | ... |
| Духовный путь | ... | ... |
| Принятие решений | ... | ... |
| Внутреннее состояние | ... | ... |

Правила:
- Содержание уникально для каждого времени
- "Что означает" — интерпретация сигнала в этой сфере
- "Как использовать" — конкретное практическое действие
- Таблица 11:11 и 22:22 должны быть принципиально разными
