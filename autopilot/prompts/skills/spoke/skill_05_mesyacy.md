## Что делает эту серию уникальной

Каждая страница — разбор энергетики конкретного месяца через его нумерологическое число.
Контент вечнозелёный — без привязки к конкретному году.
Читатель приходит с вопросом: "что несёт этот месяц и как использовать его энергию?"

Три обязательных уникальных блока на каждой странице серии:
- **Карточка месяца** — fact_row с 9 полями, уникальна для каждого месяца
- **Месяц по сферам жизни** — table с 9 строками, уникальна для каждого месяца
- **Практика месяца** — callout с вариантом practice, уникальна для каждого месяца

## Числа месяцев и их энергетика

| Месяц | Число | Энергетика |
|---|---|---|
| Январь | 1 | новое начало, воля, первый шаг |
| Февраль | 2 | партнёрство, чувства, баланс |
| Март | 3 | творчество, самовыражение, рост |

| Июнь | 6 | гармония, забота, ответственность |
| Июль | 7 | углубление, мудрость, духовный поиск |
| Август | 8 | сила, результат, материальный мир |
| Сентябрь | 9 | завершение, трансформация, отпускание |

Декабрь — 12 редуцируется до 3, но несёт двойную энергию: завершение года (9) и радость (3).

## Блок "Карточка месяца"

**Место:** сразу после lead обязательного H2, до первого H3.
**Тип блока:** `fact_row`
**Назначение:** читатель сразу видит суть энергетики месяца.

Девять обязательных полей:

| Поле | Что писать |
|---|---|
| Число месяца | порядковый номер → числовая основа |
| Энергетика | суть вибрации в 3-5 словах |
| Стихия | огонь / вода / земля / воздух |
| Архетип | образ месяца в одном слове |
| Сильные стороны | что месяц усиливает и поддерживает |
| Зоны риска | где нужна осторожность в этот период |
| Лучшее время для | конкретные действия под энергию месяца |
| Аффирмация месяца | короткая фраза-намерение (5-8 слов) |
| Совет | одна практическая рекомендация на месяц |

Содержание уникально для каждого месяца — не копировать между статьями.

Примеры:

**Январь (число 1):**
```
Число месяца → 1 — начало нового цикла
Энергетика → воля, инициатива, первый шаг
Стихия → Огонь
Архетип → Первопроходец
Сильные стороны → запуск новых проектов, постановка целей, личная воля
Зоны риска → импульсивные решения без подготовки, одиночество
Лучшее время для → начинаний, постановки намерений, планирования года
Аффирмация месяца → Я начинаю и иду первым
Совет → запишите три главных намерения на год в первую неделю января
```

**Ноябрь (число 11):**
```
Число месяца → 11 — мастер-число
Энергетика → интуиция, миссия, повышенная чувствительность
Стихия → Воздух
Архетип → Провидец
Сильные стороны → интуитивные решения, духовная работа, глубокие связи
Зоны риска → перегрузка эмоциями, тревожность, уход в себя
Лучшее время для → медитации, работы с подсознанием, важных разговоров
Аффирмация месяца → Я слышу себя и доверяю внутреннему голосу
Совет → ограничьте информационный шум — ноябрь требует тишины для важных сигналов
```

## Блок "Месяц по сферам жизни"

**Место:** сразу после lead H2 "Личный прогноз на [месяц]", до первого H3.
**Тип блока:** `table`
**Назначение:** читатель видит как энергия месяца работает в разных сферах жизни.

Три колонки, девять строк:

| Сфера | Энергия месяца | Что делать в этот период |
|---|---|---|
| Любовь | ... | ... |
| Карьера | ... | ... |
| Здоровье | ... | ... |
| Финансы | ... | ... |
| Творчество | ... | ... |
| Семья | ... | ... |
| Духовность | ... | ... |
| Обучение | ... | ... |
| Отдых | ... | ... |

Правила:
- Содержание уникально для каждого месяца
- "Энергия месяца" — как число месяца влияет на эту сферу
- "Что делать" — конкретное действие которое поддержит энергия месяца
- Таблица января и таблица сентября должны быть принципиально разными

## Блок "Практика месяца"

**Место:** в разделе "Практика [месяца]", после первого H3.
**Тип блока:** `callout` с вариантом `practice`
**Назначение:** конкретный ритуал или практика под энергетику месяца.

Правила:
- Практика уникальна для каждого месяца — не копировать шаги
- Конкретные инструкции: что делать, сколько времени, в какой момент месяца
- Аффирмация в практике совпадает с полем "Аффирмация месяца" в карточке
- Объём: 4-6 предложений

Примеры:

**Январь:**
```
В первые три дня января выделите 15 минут утром.
Запишите три намерения на год — не цели, а состояния:
кем хотите стать, а не что сделать. Перечитайте их вслух
и произнесите: «Я начинаю и иду первым.»
Возвращайтесь к этому листу в первый день каждого месяца.
```

**Сентябрь:**
```
В последнюю неделю сентября выпишите три вещи,
которые вы готовы отпустить — привычку, отношение, убеждение.
Не объясняйте зачем — просто назовите. Произнесите:
«Я завершаю и освобождаю место для нового.»
Сожгите или уберите листок — физическое действие завершает цикл.
```

```json
{
  "type": "callout",
  "variant": "practice",
  "text": "уникальная практика под конкретный месяц — 4-6 предложений"
}
```
