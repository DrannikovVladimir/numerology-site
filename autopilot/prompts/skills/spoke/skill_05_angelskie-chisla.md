## Что делает эту серию уникальной

Каждая страница — разбор конкретного ангельского числа как послания.
Читатель приходит с вопросом: "я снова вижу 111 / 1212 / 0707 — что это значит?"
Угол подачи определяется типом числа — не копируется между страницами.

Три обязательных уникальных блока на каждой странице серии:
- **Карточка числа** — fact_row с 9 полями, уникальна для каждого числа
- **Число в жизненных сферах** — table с 9 строками, уникальна для каждого числа
- **Медитация с числом** — callout с вариантом practice, уникальна для каждого числа

## Классификация типов чисел

Агент определяет тип числа из URL страницы (например /angelskie-chisla/1212/).

### Тип 1 — Тройное усиление (NNN)
Примеры: 111, 222, 333, 444, 555, 666, 777, 888, 999

Одна цифра повторяется трижды — энергия числа усилена до максимума.
Самое чистое послание: одна вибрация без примесей.
Акцент на природе базового числа и силе его усиленного сигнала.

### Тип 2 — Четверное усиление (NNNN)
Примеры: 1111, 2222, 6666, 7777, 8888, 9999

Одна цифра повторяется четырежды — ещё мощнее тройного.
Вселенная говорит особенно настойчиво и не оставляет пространства для сомнений.
Акцент на срочности послания и готовности действовать прямо сейчас.

### Тип 3 — Зеркальные (NMMN / палиндромы)
Примеры: 1221, 1331, 2112, 2332, 1441, 1551

Число читается одинаково слева направо и справа налево.
Тема: симметрия, отражение, возврат к себе, завершение цикла.
Первое число задаёт тему, второе — её отражение и завершение.

### Тип 4 — Чередующиеся (NMNM)
Примеры: 1212, 2121, 1010, 1717, 1313, 1414, 1515, 1616, 1818, 1919, 2323

Два числа чередуются в ритмическом диалоге.
Тема: баланс двух энергий, ритм, чередование фаз.
Первое число задаёт ведущую тему, второе — дополняет и уточняет.

### Тип 5 — Восходящие (NNMM)
Примеры: 1122, 1133, 1144, 1155, 2233, 2244, 2255

Первая пара передаёт силу второй паре.
Тема: нарастание, передача энергии, усиление одного другим.
Первая пара — источник, вторая — получатель и усилитель.

### Тип 6 — Комбо-строители (NMMM или NNNM)
Примеры: 1222, 1333, 1444, 1555, 2333, 0444, 0555

Одно число ведёт, три другие поддерживают и усиливают.
Тема: лидерство и поддержка, одно направляет — остальные укрепляют.
Ведущее число задаёт направление, тройное — даёт фундамент.

### Тип 7 — С нулём (0NNN / N00N / N0N0)
Примеры: 0101, 0202, 0303...0909, 0555, 0550, 1001, 2002

Ноль выступает как усилитель или символ обнуления и открытия.
Тема: пустота как возможность, начало с чистого листа, портал.
Ноль не ослабляет число рядом — он открывает пространство для него.

## Структура H2

### Обязательные H2 — присутствуют на каждой странице серии:

- **Что означает [число] в нумерологии** — суть послания, природа числа
- **[число] в любви и отношениях** — самый популярный запрос серии
- **Что делать когда видишь [число]** — практический раздел
- **Часто задаваемые вопросы** — 2-3 вопроса из семантики

### Вариативные H2 — агент выбирает 1-2 по типу числа:

| Тип | Вариативные H2 |
|---|---|
| Тройное усиление | [NNN] — когда одна энергия говорит в полный голос; Что усиливает тройное повторение |
| Четверное усиление | [NNNN] — максимальный сигнал вселенной; Почему именно сейчас так настойчиво |
| Зеркальные | Симметрия числа [NMMN] — что отражается; Возврат и завершение цикла |
| Чередующиеся | Диалог [N] и [M] в числе [NMNM]; Ритм чередования как послание |
| Восходящие | Как [N] передаёт силу числу [M]; Нарастание энергии — от [NN] к [MM] |
| Комбо-строители | Ведущее число и его усиление; Одно направляет — трое поддерживают |
| С нулём | Ноль как портал в числе [0NNN]; Пустота которая открывает путь |

### H3 под каждым H2:

Агент определяет H3 самостоятельно под тип числа и его энергетику.
2-3 H3 под каждым H2.
Не копировать H3 между разными числами серии.

## Блок "Карточка числа"

**Место:** сразу после lead первого H2, до первого H3.
**Тип блока:** `fact_row`
**Назначение:** читатель сразу видит суть этого ангельского числа.

Девять обязательных полей:

| Поле | Что писать |
|---|---|
| Числовая основа | к какому базовому числу сводится (сумма цифр) |
| Тип послания | усиление / симметрия / диалог / нарастание / портал |
| Энергетика | суть вибрации в 3-5 словах |
| Сфера влияния | где проявляется сильнее всего |
| Ангельское послание | ключевое сообщение в одном предложении |
| Частота появления | когда люди чаще всего замечают это число |
| Связь с архангелом | какой архангел связан с этим числом и почему |
| Аффирмация | короткая фраза-намерение под это число (5-8 слов) |
| Как реагировать | конкретное действие в момент встречи с числом |

Содержание уникально для каждого числа — не копировать между статьями.

Примеры:

**111:**
```
Числовая основа → 3 (1+1+1)
Тип послания → тройное усиление
Энергетика → новое начало, манифестация, мысль становится реальностью
Сфера влияния → намерения, творчество, запуск нового
Ангельское послание → ваши мысли прямо сейчас создают вашу реальность
Частота появления → в моменты принятия решений и новых начинаний
Связь с архангелом → Михаил — покровитель воли и нового пути
Аффирмация → Я создаю то, о чём думаю
Как реагировать → зафиксируйте мысль которая была в момент встречи
```

**1212:**
```
Числовая основа → 6 (1+2+1+2)
Тип послания → чередующийся диалог
Энергетика → баланс инициативы и партнёрства, ритм
Сфера влияния → отношения, совместные решения
Ангельское послание → действие и доверие должны чередоваться, не бороться
Частота появления → в периоды совместной работы или выбора между одним и другим
Связь с архангелом → Гавриил — вестник баланса и партнёрства
Аффирмация → Я действую и доверяю в равной мере
Как реагировать → спросите себя — где сейчас нарушен баланс между давать и получать
```

## Блок "Число в жизненных сферах"

**Место:** сразу после lead H2 про любовь и отношения, до первого H3.
**Тип блока:** `table`
**Назначение:** читатель видит как послание числа работает в разных контекстах жизни.

Три колонки, девять строк:

| Сфера | Послание числа | Практический шаг |
|---|---|---|
| Любовь и отношения | ... | ... |
| Карьера | ... | ... |
| Духовный путь | ... | ... |
| Здоровье и энергия | ... | ... |

| Творчество | ... | ... |
| Семья | ... | ... |

- Содержание уникально для каждого числа

- "Практический шаг" — конкретное действие которое можно сделать сегодня

## Блок "Медитация с числом"

**Место:** в разделе "Что делать когда видишь [число]", после первого H3.
**Тип блока:** `callout` с вариантом `practice`
**Назначение:** конкретная медитативная практика под энергетику числа.

Правила:
- Практика уникальна для каждого числа — не копировать шаги
- Конкретные инструкции: что визуализировать, сколько секунд, что произнести
- Аффирмация в практике совпадает с полем "Аффирмация" в карточке числа
- Объём: 4-6 предложений, не длиннее

Примеры:

**111:**
```
Остановитесь на 11 секунд. Закройте глаза и представьте
золотую единицу перед собой — она растёт и становится
вашим путём вперёд. Произнесите три раза: "Я создаю то,
о чём думаю." Запишите одно намерение которое хотите
воплотить прямо сейчас — без правок, первое что пришло.
```

**1212:**
```
Сделайте два глубоких вдоха и два медленных выдоха.
На каждом вдохе думайте о том, что хотите начать.
На каждом выдохе — о том, кому доверяете в этом пути.
Произнесите: "Я действую и доверяю в равной мере."
Спросите себя: с кем из близких стоит поговорить сегодня?
```

```json
{
  "type": "callout",
  "variant": "practice",
  "text": "уникальная практика под конкретное число — 4-6 предложений"
}
```
