## Что делает эту серию уникальной

Каждая страница — разбор конкретной пары чисел как живого союза.
Динамика пары 1+4 принципиально отличается от пары 1+9 или 5+5.
Структура и акценты статьи определяются типом пары — не копируются между страницами.

Два обязательных уникальных блока на каждой странице серии:
- **Карточка пары** — fact_row с 7 полями, уникальна для каждой пары
- **Разбор по сферам** — table с 7 строками, уникальна для каждой пары

## Классификация чисел

Агент определяет тип пары по классификации чисел:

| Группа | Числа | Суть |
|---|---|---|
| Числа силы | 1, 8 | воля, лидерство, власть |
| Числа стабильности | 4 | структура, труд, надёжность |
| Числа отношений | 2, 6 | дипломатия, забота, гармония |
| Числа творчества | 3 | самовыражение, радость, речь |
| Числа свободы | 5 | движение, поиск, перемены |
| Числа мудрости | 7, 9 | глубина, духовность, служение |
| Мастер-числа | 11, 22, 33 | особая вибрация, миссия |

| Сила + Свобода (1+5, 8+5) | напряжённый |

## Структура H2

### Обязательные H2 — присутствуют на каждой странице серии:

- **Числа [N] и [M] в нумерологии** — суть союза, природа пары
- **[N] и [M] в любви и отношениях** — романтическая динамика пары
- **Часто задаваемые вопросы** — 2-3 вопроса из семантики

### Вариативные H2 — агент выбирает 2-3 по типу пары:

| Тип пары | Вариативные H2 |
|---|---|
| зеркальный | Два [N] — сила или ловушка; Что усиливает зеркальный союз |
| конкурентный | Когда оба хотят быть первыми; Как найти баланс лидеров |
| деловой | [N] и [M] в работе и бизнесе; Партнёрство как стратегия |
| дополняющий | Как [N] и [M] восполняют друг друга; Сила в различии |
| яркий | [N] и [M] — энергия и творчество; Как не сжечь друг друга |
| напряжённый | Где [N] и [M] теряют общий язык; Как удержать союз |
| сложный | Что разделяет [N] и [M]; Путь к взаимопониманию |
| тёплый | [N] и [M] — творчество и забота; Почему этот союз держится |
| неустойчивый | [N] и [M] — свобода и близость; Как не потерять друг друга |
| гармоничный | Почему [N] и [M] понимают друг друга; Глубина без слов |
| мягкий | [N] и [M] — союз без острых углов; Сила тихой пары |
| лёгкий | [N] и [M] — искра и движение; Когда легко становится мало |
| вдохновляющий | Как [N] вдохновляет [M] и наоборот; Творчество как язык пары |
| искательский | [N] и [M] — два искателя; Свобода внутри союза |
| духовный | [N] и [M] — союз искателей; Духовное измерение пары |
| практичный | [N] и [M] в семье и быту; Как строить вместе |
| особый | Что означает мастер-число в этой паре; Миссия союза |

### H3 под каждым H2:

Агент определяет H3 самостоятельно под тип пары и конкретные числа.
2-3 H3 под каждым H2.
Не копировать H3 между разными парами.

## Блок "Карточка пары"

**Место:** сразу после lead первого H2, до первого H3.
**Тип блока:** `fact_row`
**Назначение:** читатель сразу видит суть союза этих двух чисел.

Семь обязательных полей:

| Поле | Что писать |
|---|---|
| Тип союза | деловой / романтический / дружеский / духовный — можно несколько |
| Сильная сторона | что пара даёт друг другу, их главный ресурс |
| Главное испытание | где возникает напряжение, главный вызов |
| Потенциал роста | что пара может достичь вместе |
| Риск | что разрушает эту пару чаще всего |
| Совет паре | одна конкретная практическая рекомендация |
| Совместимость | высокая / средняя / требует работы / особый случай |

Содержание уникально для каждой пары — не копировать между статьями.

Примеры для разных типов:

**Пара 1+4 (деловой):**
```
Тип союза → деловой, стратегический
Сильная сторона → единица генерирует идеи, четвёрка воплощает их в жизнь
Главное испытание → единице скучно в рамках, четвёрке тревожно без них
Потенциал роста → устойчивый бизнес или семья с чёткой структурой
Риск → единица уйдёт если четвёрка будет слишком жёсткой
Совет паре → договоритесь кто задаёт курс, а кто держит форму
Совместимость → средняя
```

**Пара 3+6 (тёплый):**
```
Тип союза → романтический, творческий
Сильная сторона → тройка приносит радость, шестёрка — тепло и заботу
Главное испытание → тройке нужна свобода, шестёрке — стабильность
Потенциал роста → творческий и уютный дом, где всегда рады гостям
Риск → тройка может чувствовать себя в ловушке заботы шестёрки
Совет паре → давайте друг другу пространство без обид
Совместимость → высокая
```

## Блок "Разбор по сферам"

**Место:** сразу после lead H2 про любовь и отношения, до первого H3.
**Тип блока:** `table`
**Назначение:** читатель видит как пара работает в разных контекстах жизни.

Три колонки, семь строк — по одной на каждую сферу:

| Сфера | Динамика пары | Что важно учесть |
|---|---|---|
| Романтика | ... | ... |
| Дружба | ... | ... |
| Работа | ... | ... |
| Семья и быт | ... | ... |
| Кризис | ... | ... |
| Деньги | ... | ... |
| Творчество | ... | ... |

Правила:
- Содержание уникально для каждой пары
- "Динамика пары" — суть взаимодействия в этой сфере, одна фраза
- "Что важно учесть" — конкретная рекомендация или предупреждение
- Таблица пары 1+4 и пары 3+6 должны быть принципиально разными
